Fields of the Lasair Object Schema
--------------------------
(1) Core Attributes
              diaObjectId: ID for this object
                       ra: Mean RA of this object
                     decl: Mean Dec of this object
              radecMjdTai: Time at which the object was at a position ra/dec ()
                     pmRa: Proper motion in right ascension (mas/yr)
                    pmDec: Proper motion in declination (mas/yr)
                 nSources: Number of diaSources associated with this diaObject
                nuSources: Number of u sources
                ngSources: Number of u sources
                nrSources: Number of u sources
                niSources: Number of u sources
                nzSources: Number of u sources
                nySources: Number of u sources
                   maxTai: Latest MJD of a diaSource
                   minTai: Earliest MJD of a diaSource
            nearbyExtObj1: Id of the closest extended DR Object by second moment-based separation.
         nearbyExtObj1Sep: Second moment-based separation of nearbyExtObj1 (unitless).
               nearbyObj1: Id of the closest nearby object.
           nearbyObj1Dist: Distance to nearbyObj1 (arcsec)
            nearbyObj1LnP: Natural log of the probability that the observed diaObject is the same as the nearbyObj1.
            nearbyLowzGal: External catalog name of the nearest low-z potential host.
         nearbyLowzGalSep: Separation distance of nearbyLowzGal. (arcsc)
               nearbyObj1: Id of the closest nearby object. ()
           nearbyObj1Dist: Distance to nearbyObj1. (arcsec)
            nearbyObj1LnP: Natural log of the probability that the observed diaObject is the same as the nearbyObj1. ()
                   absMag: Most recent absolute magnitude if host galaxy with distance available
              BBBRiseRate: Fitted Bazin or Exp rise rate
              BBBFallRate: Fitted Bazin or Exp fall rate
                  BBBTemp: Fitted Bazin temperature, kiloKelvins
                     glat: Galactic latitude
                      ebv: Extinction E(B-V) Schlegel, Finkbeiner & Davis (1998)
             mean10change: p-value for change of mean brightness: last 10 days versus 10 before that
             mean20change: p-value for change of mean brightness: last 20 days versus 20 before that
       latest_revisit_mjd: Latest revisit MJD
latest_revisit_colour_mag: Magnitude difference from latest revisit
latest_revisit_colour_bands: Bands used for revisit colour, eg g-r, u-r
latest_revisit_colour_temp: Extinction corrected effective temperature from latest revisit, kiloKelvin
                   period: Estimate of period (or NULL)
           period_entropy: Quality of period estimate (lower is better)
                    htm16: Hierarchical Triangular Mesh level 16
                timestamp: Time at which this object last modified
